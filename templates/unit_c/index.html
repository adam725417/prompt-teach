記憶體與模型大小觀念

「7B 模型需要 7GB」——這是最常見的錯誤認知！

迷思破除：7B 參數 ≠ 7GB 記憶體。實際需求還包含精度（FP16 vs INT4）、KV Cache、Context Length、Batch Size 等因素。

OpEx（運營支出）= 雲端 API

  • ✅ 無前期投資，用多少付多少
  • ✅ 快速啟動，立即驗證
  • ⚠️ 規模化後費用線性成長
  • ❗ 服務依賴廠商，有停服風險

CapEx（資本支出）= 地端部署

  • ✅ 一次性硬體投資
  • ✅ 長期使用後每次成本極低
  • ⚠️ 前期投資高，需要維運能力
  • ❗ 硬體升級需再投資
